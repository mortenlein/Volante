# Mission: GameTune

## Core Goal
Provide a transparent, truly reversible Windows 10/11 game-tweaking tool with a clean GUI and headless mode, excluding dangerous or placebo registry cleaner practices.

## Tech Stack
- Core Engine: Windows PowerShell 5.1 (Optimize.ps1)
- GUI Frontend: WPF / XAML (managed by launcher)
- Native Launcher: C++ / Rust / executable wrapper (GameTune.exe)
